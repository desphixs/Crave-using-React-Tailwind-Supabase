# Crave: A Modern Food App

## Project Narrative

You are going to build Crave, a vibrant and community-driven recipe app. You will start by designing a clean layout that showcases high-quality food photography from a list of recipes. Then, you will make the app interactive by adding a "Like" button and a real-time search tool that updates the screen as you type. Finally, you will learn how to save your favorite recipes into a personal digital box and connect your app to a live database so your data is saved forever.

## Tech Stack

- TypeScript
- TailwindCSS
- Supabase
